# Glosario

| Término | Definición |
|---|---|
| **Espacio (Space)** | Unidad alquilable: salón, sala, cancha, estudio, etc. |
| **Anfitrión (Host)** | Quien publica y opera un espacio |
| **Huésped (Guest)** | Quien reserva y paga por horas |
| **Reserva (Booking)** | Transacción confirmada de un horario en un espacio |
| **Reseña (Review)** | Calificación post-uso |
| **Listado (Listing)** | Publicación de un espacio en la plataforma |
| **Comisión** | % que Space-Peru cobra por reserva (a definir) |
| **Distrito** | División administrativa de Lima donde está el espacio |
| **Categoría** | Tipo del espacio (eventos, salas, estudios, etc.) |
| **Tarifa por hora** | Precio base que el host define |
| **Bloque mínimo** | Horas mínimas por reserva (ej: 4h en eventos) |
| **Espacio verificado** | Sello tras visita o validación documental |
