# Social

Contenido para redes. Una carpeta por plataforma. Cada post es un archivo `.md` con frontmatter.

## Estructura
```
social/
  linkedin/
  instagram/
  tiktok/
  facebook/
  twitter/
  assets/         # imágenes, videos, gifs reutilizables
  calendar.md     # calendario editorial
  README.md
```

## Frontmatter de cada post
```yaml
---
platform: instagram
date: 2026-05-10
format: carrusel  # o reel, post, story, video
status: draft     # draft / approved / published
cta: "Únete a la waitlist en planearya.com"
hashtags: [eventoslima, espacioseventos, miraflores]
---
```

## Cuándo publicar
- LinkedIn: martes/jueves AM
- Instagram: lun/mié/vie 6pm
- TikTok: cualquier día, con tendencias
- Facebook: jueves/sábado
- X: ad hoc

## Pillars de contenido
1. **Educación** (cómo elegir un espacio, checklist de eventos)
2. **Inspiración** (espacios destacados, casos)
3. **Comunidad** (testimonios host y guest)
4. **Producto** (lo nuevo de Planea Ya)
