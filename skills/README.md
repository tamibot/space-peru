# Skills

Skills propias del proyecto. Skills oficiales de Anthropic (engineering, design, marketing, etc.) están disponibles vía la lista del CLI.

## Locales

| Skill | Archivo | Cuándo |
|---|---|---|
| `space-branding` | `space-branding.md` | Aplicar tokens de branding a cualquier output visual |
| `space-copy-pe` | `space-copy-pe.md` | Generar copy en español Perú con voz de marca |
| `space-pricing-helper` | `space-pricing-helper.md` | Sugerir tarifa por hora para un espacio dado |

## Cómo se invocan
Desde Claude Code:
```
/skill <nombre>
```
o se referencian dentro de un agente.
