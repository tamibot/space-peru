# Assets

Recursos visuales reutilizables.

## Estructura sugerida (a poblar)
```
assets/
  photos/        # fotografía de espacios verificados (con permiso del host)
  mockups/       # mockups de producto en pantallas
  templates/     # plantillas para social (carrusel IG, post LinkedIn)
  icons/         # icon set custom si se hace
  og/            # imágenes Open Graph por página
```

## Reglas
- **Nunca stock genérico** (gente sonriendo en oficina, etc.). Si no hay foto real, mejor texto.
- **Foto de hosts solo con su permiso explícito por escrito** (WhatsApp + screenshot vale).
- **Tamaños estándar**:
  - Hero web: 2400×1600
  - Card: 1200×800
  - OG: 1200×630
  - IG square: 1080×1080
  - IG story / reel: 1080×1920
  - LinkedIn cover: 1584×396
- **Compresión**: WebP cuando sea web, JPG progressive para legacy.
